Política de Privacidade

Este documento descreve, de forma objetiva, como o Gabaritei trata informações durante o uso do aplicativo.

Última atualização: 14/04/2026

1. Identificação

O Gabaritei é um aplicativo desenvolvido por ZisCode para auxiliar na criação, impressão, compartilhamento, leitura e organização de gabaritos.

2. Como o app funciona

O Gabaritei foi projetado para funcionar principalmente com armazenamento local no próprio dispositivo do usuário. O app não exige conta de usuário para uso básico e não depende de cadastro para criação e leitura de provas.

3. Informações tratadas pelo app

Durante o uso, o app pode tratar:

  • descrições de provas e gabaritos cadastrados pelo usuário;
  • número de questões, alternativas e respostas definidas;
  • imagens capturadas pela câmera para leitura de gabaritos;
  • arquivos gerados pelo próprio usuário, como PDFs e imagens de compartilhamento;
  • dados técnicos de uso e de diagnóstico para estabilidade do app.

4. Armazenamento local

As informações de provas, gabaritos e configurações do aplicativo são armazenadas localmente no dispositivo. Em condições normais de atualização do app, esses dados devem ser preservados.

O usuário é responsável por manter cópia, controle e conferência dos materiais que gerar ou utilizar no aplicativo.

5. Uso da câmera

O app solicita acesso à câmera para capturar imagens de gabaritos e permitir a leitura das marcações. A câmera é usada apenas para execução dessa funcionalidade.

6. Compartilhamento pelo usuário

O Gabaritei pode gerar e compartilhar, por ação do próprio usuário:

  • arquivos PDF;
  • imagens com QR Code;
  • provas preenchidas ou em branco;
  • provas importáveis por QR Code.

Esses compartilhamentos só ocorrem quando o usuário decide executá-los. O usuário é responsável pelo conteúdo que escolher compartilhar e pelo destino desse material.

7. QR Code e permissões de edição

O app permite compartilhar provas por QR Code com regras definidas pela origem, inclusive com permissão de edição habilitada ou desabilitada para o dispositivo receptor. O usuário que compartilha o conteúdo é responsável pela escolha dessas permissões.

8. Firebase Analytics e Crashlytics

Para melhorar estabilidade, desempenho e experiência de uso, o Gabaritei utiliza serviços do Firebase, incluindo Firebase Analytics e Firebase Crashlytics.

Esses serviços podem coletar dados técnicos e de uso, como:

  • eventos de navegação e uso de recursos;
  • versão do app;
  • modelo do dispositivo;
  • sistema operacional;
  • informações técnicas sobre falhas, exceções e travamentos;
  • identificadores técnicos necessários para funcionamento dos serviços do Firebase.

Esses dados são utilizados para monitoramento, correção de erros, análise de uso e evolução do aplicativo.

O app não tem como finalidade enviar para esses serviços, por padrão, imagens da prova, conteúdo integral de PDFs, respostas completas de alunos ou dados pessoais sensíveis inseridos pelo usuário.

9. Dados pessoais eventualmente inseridos pelo usuário

O aplicativo não foi projetado como plataforma de cadastro de dados pessoais. Porém, o usuário pode, por sua própria iniciativa, inserir descrições, imagens ou materiais que contenham informações relacionadas a terceiros, como alunos.

Nesses casos, o usuário é o responsável pelo conteúdo inserido, pelo uso adequado desse material e pela verificação de sua base legal, quando aplicável.

10. Exclusão de dados

Os dados locais podem ser excluídos pelo próprio usuário a partir das funcionalidades do app ou pela remoção do aplicativo do dispositivo, o que pode apagar os dados armazenados localmente.

11. Alterações desta política

Esta Política de Privacidade pode ser atualizada futuramente para refletir alterações legais, técnicas ou funcionais do aplicativo. Quando necessário, o app poderá solicitar nova ciência do usuário.

12. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato pelo e-mail ziscodeapps@gmail.com.
